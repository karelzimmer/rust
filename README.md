# rust

Mijn Rust-programma's.
Voor (zelf)studie. Werk in uitvoering.

## Werkwijze

```console
cd ~/rust
cd hello-world
```

Edit main.rs:

```rust
fn main() {
    println!("Hello, world!");
}
```

Compile:

```console
rustc main.rs
```

Run:

```console
./main
```

Output:

```console
Hello, world!
```
